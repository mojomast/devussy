### Before You Deploy

1. **Pull Latest Code**
   ```bash
   cd devussy/devussy-web
   git pull origin adaptive-llm-clean
   ```

2. **Configure Environment**
   ```bash
   cp .env.example .env
   # Edit .env with your secrets:
   # - REQUESTY_API_KEY
   # - STREAMING_SECRET
   # - NEXT_PUBLIC_IRC_WS_URL (optional, has default)
   # - NEXT_PUBLIC_IRC_CHANNEL (optional, has default)
   ```

3. **Build and Deploy**
   ```bash
   docker-compose -f docker-compose.prod.yml down
   docker-compose -f docker-compose.prod.yml build --no-cache
   docker-compose -f docker-compose.prod.yml up -d
   ```
